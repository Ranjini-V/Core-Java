METHOD OVERRIDING

- Possible only in multiple classes [Inheritance]
- Signature of the method [ method name, return type, parameters, access] should not be changed bt the method body can be changed
- Overriing is an INHERITANCE concept


METHOD OVERLOADING

- Possible in single [without Inheritance]/ multiple classes [Inheritance]
- Signature of the method [not the name but the return type, number of parameters, data type of parameters] shoukd be changed
- Overloading is a POLYMORPHISM concept
